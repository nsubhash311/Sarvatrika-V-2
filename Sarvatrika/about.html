Project Description

Machine Learning models are capable of processing huge amounts of data. As the amount and type of data varies, the algorithms will also vary. Algorithm selection is a manual and tedious task. Our motto is to give an universal solution for uneducated people who can easily use it. We also provide user friendly and simple steps to predict.

Each and every prediction is dependent on the dataset. Thus no constant algorithm is used. As of now, the algorithms are chosen after analysing the datasets.

Since it is a tedious process here we introduce a universal prediction named as Sarvatrika which analyzes the dataset and chooses a perfect machine learning prediction algorithm. This idea of introducing a universal prediction method is called Sarvatrika.

Universal

It deals with all kinds of dataset and works as a universal solution for all data.

Automation

It cleans the dataset and perform Pre-processing by automation even when the dataset is in millions.

Prediction

It analyzes the dataset and chooses a perfect machine learning algorithm which predicts values.
